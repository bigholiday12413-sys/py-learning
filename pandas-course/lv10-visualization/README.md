# pandas講座 Lv10 - グラフ化の基本

## テーマ

集計した数字は、表よりグラフの方が一瞬で伝わることが多い。
pandasに組み込まれた `.plot()` と、その裏で動く matplotlib の最低限を扱う。

## 動かし方

```bash
cd pandas-course/lv10-visualization
python main.py
```

`output/` に5つのグラフ（png画像）が生成される。生成された画像を開いて確認する。

## 最重要ポイント: groupbyの結果はそのままグラフになる

```python
by_category = orders.groupby("category")["subtotal"].sum()
by_category.plot(kind="bar")
```

Lv05のgroupbyの結果はSeries/DataFrameなので、**そのまま `.plot()` が使える**。
「集計する」と「見せる」がほぼシームレスにつながっているのがpandasの強み。

## 学べること

| トピック | ひとことで言うと |
|---------|----------------|
| `.plot(kind="bar")` | 棒グラフ（項目ごとの比較） |
| `.plot(kind="line")` | 折れ線グラフ（時間の推移） |
| `.plot(kind="pie")` | 円グラフ（構成比。項目5〜6個まで推奨） |
| `.plot(kind="bar", stacked=True)` | 積み上げ棒グラフ（複数系列＋合計） |
| `plt.savefig(path)` | グラフを画像ファイルとして保存 |
| `plt.tight_layout()` | ラベルがはみ出さないよう自動調整 |
| 日本語フォント設定 | 文字化け（□□□）対策 |

## ビジネスで使うグラフの選び方

| 伝えたいこと | グラフ |
|-------------|--------|
| 項目ごとの比較（カテゴリ別売上など） | 棒グラフ |
| 時間の推移（月次売上の変化） | 折れ線グラフ |
| 全体に対する割合 | 円グラフ（項目が多いなら棒グラフの方が読みやすいことも多い） |
| 複数系列の内訳＋合計（月次×カテゴリ） | 積み上げ棒グラフ |

## 日本語フォントの文字化けについて

matplotlibはデフォルトで日本語フォントを持っていないため、何もしないとタイトルや
軸ラベルの日本語が `□□□` になる。`main.py` では利用可能なフォントを自動で探して
設定しているが、**環境によっては日本語フォントが1つも見つからない**ことがある
（特にLinuxサーバー環境）。その場合の対処:

- Windows: `Meiryo` や `Yu Gothic` が標準で入っている
- macOS: `Hiragino Sans` が標準で入っている
- Linux: `sudo apt install fonts-noto-cjk` 等で日本語フォントを追加する
