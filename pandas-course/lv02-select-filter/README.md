# pandas講座 Lv02 - 抽出とフィルタリング（絞り込み）

## テーマ

「条件に合う行だけを取り出す」は、pandasを使う理由の半分を占める操作。
Excelの「フィルタ」機能をコードで再現する感覚で読み進める。

## 動かし方

```bash
cd pandas-course/lv02-select-filter
python main.py
```

## 核心: ブールインデックス

```python
df["quantity"] >= 2
```

これは「各行が条件を満たすか」の **True/False が並んだ列** を作る。それを `df[...]` に
渡すと、**True の行だけが残った表** になる。

```python
result = df[df["quantity"] >= 2]
```

Excelでフィルタボタンを押す代わりに、条件を1行のコードで書いていると考えるとよい。

## 複数条件の書き方（要注意ポイント）

```python
df[(df["quantity"] >= 2) & (df["unit_price"] >= 2000)]   # かつ
df[(df["category"] == "文房具") | (df["category"] == "インテリア")]  # または
df[~(df["category"] == "文房具")]                          # 否定
```

- Python の `and`/`or` ではなく **`&`/`|`**
- 各条件は **必ず `()` で囲む**（囲み忘れは最頻出のエラー）

## 学べること

| トピック | ひとことで言うと |
|---------|----------------|
| `df[df["列"] >= 値]` | ブールインデックスによる絞り込み |
| `&` / `\|` / `~` | かつ・または・否定（各条件を`()`で囲む） |
| `.isin([...])` | 「複数候補のいずれか」を簡潔に書く |
| `.value_counts()` | 各値の出現回数を数える（集計の入り口） |
| `df.loc[条件, ["列1","列2"]]` | 行の絞り込みと列選択を同時に |
| `df.query("条件式")` | 文字列で条件を書く。Excelフィルタに近い読みやすさ |
| `query()` の `@変数` | 外部のPython変数を条件式に埋め込む |
