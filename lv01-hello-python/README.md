# Lv01 - Hello Python: はじめての Python

## テーマ

Python の基本文法を一通り体験する。
変数・型・文字列・リスト・辞書・関数・条件分岐・ループなど、
プログラミングの土台になる要素をゼロから学ぶ。

## 動かし方

```bash
cd py-learning/lv01-hello-python
python main.py
```

※ Python 3.10 以上を推奨。`python --version` で確認できる。

## 学べること

| # | トピック | ひとことで言うと |
|---|---------|----------------|
| 1 | 変数宣言 | `名前 = 値` と書くだけ |
| 2 | データ型 (str, int, float, bool, None) | 文字列・整数・小数・真偽値・「値なし」 |
| 3 | f-string | 文字列に `{変数}` を埋め込む |
| 4 | リスト (list) | 複数の値を順番に並べる `[]` |
| 5 | 辞書 (dict) | キーと値のペアで持つ `{}` |
| 6 | 関数 (def / lambda) | 処理に名前を付けて再利用する |
| 7 | 条件分岐 (if/elif/else) | 条件によって処理を分ける |
| 8 | for / while ループ | 繰り返し処理 |
| 9 | Truthy / Falsy | 「空」の値は偽として扱われる |
| 10 | 型ヒント (Type Hints) | 引数や戻り値の型をコードに書き残す |
| 11 | print フォーマット | 画面出力のいろいろな書き方 |

## 読む順番
