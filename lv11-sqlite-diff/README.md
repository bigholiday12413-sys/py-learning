# Lv11 - SQLiteでデータ蓄積と差分検知【発展編】

## テーマ

スクレイピング結果を実行のたびにデータベースへ蓄積し、
**「前回実行時から何が変わったか」** を自動検出する。

新着検知・価格監視・在庫監視は業務自動化の定番ニーズ。
「毎回全件取ってCSVを見比べる」から「変わったものだけ報告する」への進化がこのレベル。

## SQLite とは

- Python **標準ライブラリ (`sqlite3`) だけで使える**データベース。追加インストール不要
- サーバー不要。`books.db` のような **ファイル1つ** で完結する
- それでいて SQL という業界標準の問い合わせ言語がフルに使える
- スマホアプリやブラウザの内部でも使われている、世界で最も使われているDB

## 動かし方

```bash
cd lv11-sqlite-diff

# 1回目: 初回スナップショットが保存される
python main.py

# 2回目: サンプルデータが少し変化し、差分レポートが出る
python main.py
```

2回目の実行で以下が検出される（サンプルデータに仕込んである）:

```
[新規]     Sapiens: A Brief History
[取扱終了] Soumission
[値下げ↓]  Tipping the Velvet: £53.74 → £49.99
```

リセットしたいときは `books.db` を削除する。

## 学べること

| トピック | ひとことで言うと |
|---------|----------------|
| `sqlite3.connect()` | DBファイルを開く（無ければ作る） |
| `CREATE TABLE` / `INSERT` / `SELECT` | SQL の基本3命令 |
| `?` プレースホルダ | SQLに値を安全に渡す（**f-string でSQLを組み立てない！**） |
| `executemany()` | 複数行の一括INSERT |
| `fetchone()` / `fetchall()` | 検索結果の取り出し |
| `LEFT JOIN` / `GROUP BY` | テーブルの結合と集計 |
| 辞書のキー集合演算 `-` `&` | 差分検知のコアロジック |
| 辞書内包表記 | `{k: v for ...}`（リスト内包表記の辞書版） |

## テーブル設計

```
snapshots（実行履歴）          books（書籍データ）
┌────┬────────────┐      ┌────┬─────────────┬───────┬───────┐
│ id │ created_at │ ←──┐ │ id │ snapshot_id │ title │ price │
├────┼────────────┤    └─┼────┼─────────────┼───────┼───────┤
│ 1  │ 07-02 09:00│      │ 1  │ 1           │ ...   │ 51.77 │
│ 2  │ 07-02 10:00│      │ 2  │ 1           │ ...   │ 53.74 │
└────┴────────────┘      │ .. │ 2           │ ...   │ ...   │
```

「1回の実行 = 1スナップショット」とし、書籍は `snapshot_id` でどの実行分かを紐づける。
この形にしておくと、直前との比較だけでなく「1週間前との比較」も同じ仕組みでできる。

## DBを覗く

作成された `books.db` の中身は、以下のどちらかで確認できる:

```bash
# 方法1: Python の対話モードで

>>> conn = sqlite3.connect("books.db")
>>> conn.execute("SELECT * FROM snapshots").fetchall()

# 方法2: GUIツール「DB Browser for SQLite」（無料）で books.db を開く
```

## ⚠ セキュリティの注意（実務で必ず守ること）

SQL に値を埋め込むときは、**必ず `?` プレースホルダを使う**。

```python
# ❌ 絶対にやらない: 値の中に ' が入ると壊れる & SQLインジェクションの温床
conn.execute(f"SELECT * FROM books WHERE title = '{title}'")

# ✅ 正しい書き方
conn.execute("SELECT * FROM books WHERE title = ?", (title,))
```
