# このリポジトリについて

## 目的

Python未経験者が、**最短ルートで Playwright によるブラウザ自動操作 exe** を作れるようになること。
プログラミング自体が初めてでも進められるよう、Lv1 で Python の基礎文法から丁寧に解説する。

## 最終ゴール

- Playwright でWebサイトにログインし、画面をスクレイピングし、ボタンをクリックする
- その処理を PyInstaller で **exe ファイル** にパッケージングする
- 設定ファイル（JSON/TOML）を外出しして、コード変更なしで対象サイトや操作を切り替えられるようにする

## よく使う Python の道具箱（先取りリスト）

コースを進める中で繰り返し登場する道具たち。忘れたらここに戻って確認する。

| 道具 | 何をするもの | 初登場 |
|------|-------------|--------|
| `print()` / `input()` | 画面出力・キーボード入力 | Lv1 |
| f-string `f"{x}"` | 文字列に値を埋め込む | Lv1 |
| リスト `[]` / 辞書 `{}` | データをまとめて持つ | Lv1 |
| `if` / `for` / `while` | 条件分岐・繰り返し | Lv1 |
| `def` / 型ヒント | 関数定義・型の注釈 | Lv1 |
| `try` / `except` | エラー処理 | Lv1 |
| `with open(...)` | ファイルの安全な読み書き | Lv2 |
| `csv` / `pathlib` | CSV処理・パス操作 | Lv2 |
| `class` / `@dataclass` | データと処理をまとめる | Lv3 |
| `import` / venv / pip | モジュール分割・環境と依存の管理 | Lv3 |
| `requests` / BeautifulSoup | HTTP取得・HTML解析 | Lv4 |
| Playwright | ブラウザ自動操作 | Lv5 |
| `logging` | 実務向けのログ出力 | Lv6 |
| `json` / PyInstaller | 設定ファイル・exe化 | Lv8 |
